## Usage
#### Add a task:	
	1. Left click the **Add Task** bar
	2. Type the task's name
	3. Press **enter** to create the task

#### Complete a task:
	Left click the check box next to the task

#### View saved tasks:
	Navigate to .vscode/TaskList.json
